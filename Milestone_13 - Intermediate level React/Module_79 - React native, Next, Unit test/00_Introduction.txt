১. রিএক্ট নেটিভ: 

আজকের একদম প্রথমেই আছে React Native নিয়ে প্রাথমিক ধারণা। প্লাস কিভাবে তুমি সহজ এবং সিম্পলভাবে একটা React Native এর এপ স্টার্ট করতে পারো। এরপরে React Native রিলেটেড কিছু বেসিক কম্পোনেন্ট নিয়ে আলোচনা করা হয়েছে। কিভাবে props, state, hooks কাজ করে। কিভাবে ডাইনামিকভাবে ডাটা লোড করে দেখা যায়। এমনকি কিভাবে তুমি তোমার মোবাইলে এপটা দেখতে পারো। এই অংশ দেখানোর একটা উদ্দেশ্য হচ্ছে তোমরা যারা React নিয়ে কাজ করতে যাবে। তোমাদের মধ্যে অল্প কয়েকজনকে তাদের কোম্পানি React Native নিজে কাজ করতে বললে তুমি যেন ভড়কে না যাও। 



React নেটিভ নিয়ে কাজ করতে বললে এই তিনটা কাজ করবে 

১. ইউটুবে গিয়ে লাস্ট ৬ মাস বা সর্বোচ্চ ১ বছরের মধ্যে পাবলিশ হওয়া কোন একটা ভিডিও দেখে দেখে একটা প্রজেক্ট করে ফেলো। দুই তিনটা করে ফেলতে খুবই ভালো। এই কাজ করতে তোমার সময় লাগবে সর্বোচ্চ ২ দিন। 

২. React নেটিভ এর অফিশিয়াল ডকুমেন্টেশন পড়তে সময় লাগবে ২ দিন। 

৩. অফিসের কাজের জন্য React native এ কি কি করতে হবে সেটা জেনে আরো ১দিন সময় দিলে 



ব্যস মোটামুটি ৫ দিন সিরিয়াসলি সময় দিলে তুমি কিন্তু React Native এ কাজ করার জন্য প্রাথমিকভাবে রেডি হয়ে যাবে। এরপর কাজ করতে গেলে স্বাভাবিকভাবেই আটকে যাবে। সেগুলা ধরে ধরে সমাধান করতে করতেই তুমি সামনে এগিয়ে যাবে। 



.



২. নেক্সট জেস: 

একটু ছোট করে হালকার উপর দিয়ে নেক্সট জেস সম্পর্কে ওভারভিউ দেয়া হয়েছে। এইটা কেন ইউজ করা হয়। কি কাজে লাগে সেটা বলা হয়েছে। এছাড়াও কিভাবে সিম্পলভাবে নেক্সট জেস দিয়ে রাউট তৈরি করে ফেলা যায়। কিভাবে নেক্সট জেস এর ভিতরে ডাটা লোড করতে হয়। স্পেশাল ফাংশন দিয়ে। তারপর দেখানো হয়েছে কিভাবে নেক্সট জেস এর মধ্যে ডাইনামিক রাউট তৈরি করা হয়। মূলত হালকা করে একটু ওভারভিউ দেয়া হয়েছে। 



.
